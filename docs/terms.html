What this is

slopstop is a Chrome extension that scores social media posts for likely AI-generation using heuristic pattern matching and an optional AI-detector model. It is provided as-is, with no warranty.

Limits of detection

AI detection is inherently imperfect. Scores are estimates, not verdicts. False positives and false negatives are expected. Do not use this extension to make accusations, hiring decisions, or any consequential judgments about specific individuals.
