What it does

Real-time scoring

Each post and comment gets a colored badge showing how likely it is to be AI-generated, on a 0–100 scale.

Reasoning, not just verdicts

Hover any badge to see exactly which patterns triggered the score: stock phrases, em-dashes, emoji-bullet structure.

Privacy by default

The heuristic badge runs entirely in your browser — no post text leaves your machine. The optional AI detector only sends text when you ask it to.

Optional AI detector

Want a second opinion? Click Run AI detector on any post to score it with an ML model. Free for 5 checks a day when signed in; paid tiers raise the daily limit.

How it works

  1. 1
    Install the extension and open LinkedIn or Substack.
  2. 2
    Scroll your feed. Every post and comment gets a small score badge in the corner.
  3. 3
    Hover any badge to see why it scored that way — em-dash density, stock phrases, structural tells.
  4. 4
    Want more certainty? Click Run AI detector in the tooltip to score the post with an ML model — 5 free checks a day when signed in, more on paid tiers.
  5. 5
    Treat it as a hint. Heuristics aren't proof. Polished writers will trip false positives.

Install slopstop

Add it to Chrome from the Chrome Web Store. The heuristic badges are free, with no account required.
