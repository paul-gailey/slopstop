A Chrome extension that scores every LinkedIn and Substack post and comment for AI tells, live as you scroll. Free.

Install on Chrome — free See how it works
No account required · Runs on-device · Works on LinkedIn and Substack

What it does

Real-time scoring

Each post and comment gets a colored badge showing how likely it is to be AI-generated, on a 0–100 scale.

Reasoning, not just verdicts

Hover any badge to see exactly which patterns triggered the score: stock phrases, em-dashes, emoji-bullet structure.

Privacy by default

Heuristics run entirely in your browser. No post text leaves your machine.

How it works

  1. 1
    Install the extension and open LinkedIn or Substack.
  2. 2
    Scroll your feed. Every post and comment gets a small score badge in the corner.
  3. 3
    Hover any badge to see why it scored that way — em-dash density, stock phrases, structural tells.
  4. 4
    Treat it as a hint. Heuristics aren't proof. Polished writers will trip false positives.

Install

The extension is currently in development and available for sideloading via "Load unpacked" in Chrome.

A Chrome Web Store listing is coming soon. Sign up in the extension popup to be notified when Pro features launch.

Want to self-host with your own AI-detector model? slopstop is open source — view the repo on GitHub.
